# 💻 Hands On - Estruturas de Controle II (Java)

Este repositório contém a resolução de exercícios práticos utilizando **Java**, com foco em estruturas de controle, manipulação de dados e lógica de programação.

---

## 📌 Exercícios Implementados

### 🎨 1. Desenho em ASCII

Programa que imprime um desenho no console utilizando caracteres especiais.

```
 +"""""+                                                 
[| o o |]                                                
 |  ^  |                                                 
 | '-' |                                                 
 +-----+
```

---

### 🌍 2. Cálculo de Distância entre Dois Pontos na Terra

Implementação da fórmula matemática para calcular a distância entre duas coordenadas geográficas (latitude e longitude).

📥 **Dados utilizados:**

* Latitude 1: 25°
* Longitude 1: 35°
* Latitude 2: 35.5°
* Longitude 2: 25.5°
* Raio da Terra: 6371,01 km

📤 **Saída esperada:**

```
A distância entre esses pontos é: 1480.0848451069087 km
```

---

### 🔤 3. Contador de Caracteres

Programa que analisa uma string digitada pelo usuário e contabiliza:

* Letras
* Números
* Espaços
* Outros caracteres

---

### ❓ 4. Questão de Múltipla Escolha

Sistema interativo que:

* Apresenta uma pergunta com 5 alternativas (a–e)
* Permite até **3 tentativas**
* Informa se a resposta está correta ou incorreta
* Exibe em qual tentativa o usuário acertou (se acertar)

---
